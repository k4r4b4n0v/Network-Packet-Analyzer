# Network Packet Analyzer

**Анализатор сетевого трафика** – программа с графическим интерфейсом для просмотра и анализа PCAP-файлов.

## 🚀 Возможности

- Открытие файлов `.pcap` и `.pcapng`
- **Drag & Drop** – перетащите файл в окно
- Для каждого пакета отображается:
  - Время захвата
  - Протокол (TCP/UDP)
  - IP-адреса и порты источника/назначения
  - **TLS/SSL-рекорды** (тип, версия, длина)
  - Первые 50 байт полезной нагрузки в HEX и ASCII
- Поддержка **тёмной темы**
- **Русский и английский** язык интерфейса
- Сохранение результатов анализа в TXT-файл

## 🛠️ Технологии

- Python 3.7+
- Scapy — чтение и разбор PCAP-файлов
- PyQt5 — графический интерфейс

## 📦 Установка и запуск

```bash
# 1. Установите зависимости
pip install scapy PyQt5

# 2. Запустите программу
python analyzer.py
```
